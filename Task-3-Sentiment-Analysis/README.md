# Task 3 - Sentiment Analysis on Amazon Reviews

## Project Overview
This project is a Machine Learning based Sentiment Analysis system built using Python and Scikit-learn.

The model analyzes Amazon product reviews and predicts whether the sentiment of the review is:

- Positive 😊
- Negative 😡

The project uses Natural Language Processing (NLP) techniques such as:
- Text Cleaning
- Stopword Removal
- TF-IDF Vectorization
- Logistic Regression

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLTK
- Scikit-learn

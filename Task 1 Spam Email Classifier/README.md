# 📧 Task 1: Spam Email Classifier

## 📌 Description
This project was developed as part of the **Kodbud Tech-Driven Artificial Intelligence Internship Program**.

The objective is to build a Machine Learning model that classifies SMS/Email messages as **Spam** or **Ham (Legitimate)** using Natural Language Processing (NLP) techniques.

---

## 🚀 Features

- ✅ **Exploratory Data Analysis (EDA)**  
  Visualized class distribution and analyzed dataset balance.

- ✅ **Text Preprocessing**  
  Cleaning, tokenization, and preparation of text data.

- ✅ **TF-IDF Vectorization**  
  Converted text into numerical feature vectors based on word importance.

- ✅ **Multinomial Naive Bayes Classifier**  
  Trained a probabilistic model optimized for text classification.

- ✅ **Predictive System**  
  Custom function to test new messages for spam detection.
